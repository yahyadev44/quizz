# Penambah Wawasan 🧠

Aplikasi quiz interaktif modern berbasis **Vue.js + Express.js** dengan tampilan elegant, responsive, dan realtime correction.

---

## ✨ Fitur

- 🌍 Quiz Global dari API Open Trivia DB
- 🇮🇩 Quiz Indonesia (JSON Lokal)
- 🔄 Auto Translate ke Bahasa Indonesia
- ✅ Koreksi jawaban realtime
- 📈 Skor otomatis
- 📱 Responsive Mobile Design
- ⚡ Loading animation
- 🎨 Modern UI (TailwindCSS)

---

# 🛠️ Tech Stack

## Frontend
- Vue 3
- Vite
- TailwindCSS
- Axios

## Backend
- Express.js
- Axios
- CORS

# 🚀 Instalasi

## 1. Clone Repository

```bash
git clone https://github.com/USERNAME/REPOSITORY.git
```

---

# ⚙️ Backend Setup

Masuk ke folder backend:

```bash
cd backend
```

Install dependency:

```bash
npm install
```

Jalankan backend:

```bash
node index.js
```

Backend berjalan di:

```text
http://localhost:5000
```

---

# 🎨 Frontend Setup

Masuk ke folder frontend:

```bash
cd frontend
```

Install dependency:

```bash
npm install
```

Jalankan frontend:

```bash
npm run dev
```

Frontend berjalan di:

```text
http://localhost:5173
```
